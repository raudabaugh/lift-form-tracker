Phone Setup

  1. Open this URL in Safari (iPhone) or Chrome (Android) — not a QR scanner
  2. HTTPS is required for camera access — GitHub Pages provides this automatically
  3. Tap a stage, wait for model load (5–15 s on first visit), then tap Start Camera
  4. Prop the phone at floor level, 3–4 ft away, facing you. Landscape orientation gives a wider field of view
  5. Stage 3: tap ↓ Log N after your set to export the session JSON
